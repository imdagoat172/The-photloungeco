# The Photo Lounge ATL

A full-stack website for photo booth rental services.

## Tech Stack
- Frontend: Angular
- Backend: Python Flask
- Database: SQLite

## Features
- Multi-page website with Home, Services, FAQ, Contact
- Contact form to submit inquiries
- Services for Wedding, Birthday, Corporate Events (all starting at $795)

## Setup

### Backend
1. cd backend
2. source venv/bin/activate
3. python app.py

Runs on http://localhost:5000

### Frontend
1. cd frontend
2. npm install
3. ng serve

Runs on http://localhost:4200
